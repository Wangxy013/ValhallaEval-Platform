# 第1章 快速开始

本章通过一个完整示例，带你在 5 分钟内完成第一次 Prompt 效果对比评测。

**目标**：验证为答题 Prompt 增加格式要求后，输出格式的合规率是否提升。

---

## 前提条件

- 系统已成功启动（访问 http://localhost:3000 可打开界面）
- 你有一个支持 OpenAI 兼容接口的大模型 API Key（如豆包、星火、OpenAI 等）

---

## Step 1：配置模型

1. 点击左侧菜单 **「模型管理」**
2. 点击 **「新建模型」**，填写：
   - **模型名称**：豆包 Pro（自定义，便于识别）
   - **提供商**：doubao
   - **API URL**：`https://ark.cn-beijing.volces.com/api/v3`
   - **API Key**：你的 sk-xxx Key
   - **模型 ID**：你的 endpoint id（如 ep-xxx）
3. 点击 **「保存」**

> 模型配置成功后，在本页面可看到新增的模型卡片。

---

## Step 2：录入两版 Prompt

1. 点击左侧菜单 **「Prompt管理」**
2. **第一版（优化前）**：点击「新建 Prompt」
   - 名称：`答题助手`
   - 版本：`v1`
   - 内容：`你是一名专业的数学教师，请回答学生的问题。`
   - 保存
3. **第二版（优化后）**：再次点击「新建 Prompt」
   - 名称：`答题助手`
   - 版本：`v2`
   - 内容：`你是一名专业的数学教师，请回答学生的问题。\n\n必须按以下格式输出：\n【答案】xxx\n【解析】xxx`
   - 修改说明：`增加格式约束`
   - 保存

---

## Step 3：创建测试数据

1. 点击左侧菜单 **「测试数据」**
2. 点击「新建数据集」，名称填 `数学题样本`，保存
3. 点击新建的数据集，进入详情页
4. 添加 3-5 条测试数据，例如：
   - `已知三角形三边长分别为3、4、5，求各角的角度。`
   - `一元二次方程 x²-5x+6=0 的解是什么？`
   - `等差数列首项为2，公差为3，求第10项。`

---

## Step 4：创建评测任务

1. 点击左侧菜单 **「评测任务」**，再点击「新建任务」
2. 按向导填写：

**第1步 基本信息**
- 任务名称：`答题助手格式优化效果验证`
- 评测类型：选 **「Prompt 对比测试」**

**第2步 选择模型**
- 勾选刚才配置的「豆包 Pro」

**第3步 选择 Prompt**
- 同时勾选 `答题助手 (v1)` 和 `答题助手 (v2)`

**第4步 测试数据**
- 选择「数学题样本」数据集
- 全选所有测试数据

**第5步 校验与评估**
- 点击「添加检查点」，填写：
  - 检查点名称：`格式规范`
  - 校验标准：`输出必须同时包含【答案】和【解析】两个标签`
- 评估模式选 **「自动评估」**
- 点击「创建任务」

---

## Step 5：执行与校验

任务创建后自动跳转到任务结果页。

1. 点击 **「开始执行」** — 系统开始批量调用 LLM，状态变为「运行中」
2. 等待推理完成（状态变为「已完成」），页面每 3 秒自动刷新
3. 点击 **「运行校验」** — LLM 裁判自动判断每条输出是否符合格式标准
4. （可选）点击 **「自动评估」** — LLM 对每条输出打 0-10 分

---

## Step 6：查看结果

**总览页**

- 查看「验证点对比（Prompt 版本横向比较）」矩阵：
  - 行 = 格式规范（你设置的校验点）
  - 列 = v1、v2
  - 每格显示通过率 + 对比第一列的 `±N%`
- 如果 v2 的格式规范通过率显著高于 v1（如 `+40%`），说明格式优化生效

**明细页**

- 展开任意一行测试数据
- 左侧卡片 = v1 的输出，右侧卡片 = v2 的输出
- 底部「对比差异」展示该条数据中哪些校验点改善了，哪些一致
